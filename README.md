# asistencias_web
 Este trabajo es un sistema web simple para registrar las asistencias de los alumnos del Colegio Santa María.
 Permite que los preceptores marquen presentes, ausentes o llegadas tarde, y que el administrador pueda ver los reportes por fecha.
 El proyecto fue hecho con HTML, CSS, JavaScript y usa LocalStorage para guardar los datos sin base de datos externa.
# framework elegido:
 Usamos Bulma, un framework CSS que facilita el diseño de páginas web responsivas y modernas sin necesidad de escribir mucho código.
 Nos gustó porque es liviano, fácil de usar y tiene un estilo limpio que se adapta bien al proyecto.
